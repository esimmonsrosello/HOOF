Instructions to Set Up and Run a Python Project with Streamlit (macOS/Linux)

1. Open Terminal and navigate to your project folder:
cd "/path/to/your/project/folder"

2. Create a virtual environment (only needed the first time):
python3 -m venv .venv

3. Activate the virtual environment:
source .venv/bin/activate
(Note: You should see "(.venv)" at the start of the terminal prompt.)

4. Install dependencies from requirements.txt:
pip3 install -r requirements.txt

5. Run the Streamlit app:
streamlit run HOOF.py
(A browser window should open automatically.)

6. Stop the app when done:
Press CTRL + C in the terminal.

7. Deactivate the virtual environment:
deactivate



Instructions to Set Up and Run a Python Project with Streamlit (Windows)

1. Open PowerShell or Command Prompt and navigate to your project folder:
cd "C:\path\to\your\project\folder"

2. Create a virtual environment (only needed the first time):
python -m venv .venv

3. Activate the virtual environment:

In PowerShell:
.venv\Scripts\Activate.ps1

In Command Prompt:
.venv\Scripts\activate.bat

(Note: You should see "(.venv)" at the start of the prompt in PowerShell.)

4. Install dependencies from requirements.txt:
pip install -r requirements.txt

5. Run the Streamlit app:
streamlit run HOOF.py
(A browser window should open automatically.)

6. Stop the app when done:
Press CTRL + C in the terminal.

7. Deactivate the virtual environment:
deactivate
